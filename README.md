# QR Code Component - Frontend Mentor

## 📌 Description

This project is a responsive QR code component built with HTML and CSS.
It is a solution to a Frontend Mentor challenge aimed at improving front-end development skills.

## 🚀 Features

* Responsive design (mobile & desktop)
* Clean UI card layout
* SEO optimized
* Accessible structure (ARIA, semantic HTML)

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox)
* Responsive Design
